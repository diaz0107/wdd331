Using SVG Attributes As CSS Properties

Some SVG element attributes have equivalent presentation attributes. We can use CSS to set the position of rect, circle, and ellipse elements.

Although units are optional for SVG attributes, they're required when using CSS.

The last two demos require a Chromium-based browser, such as Google Chrome or Microsoft Edge.

(Note: SVG also supports the use of the style attribute for elements.)
