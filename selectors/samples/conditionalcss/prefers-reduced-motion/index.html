Using prefers-reduced-motion

This demo may cause dizziness or seizures if you have a vestibular or seizure disorder.

Animations and transitions can, for some people, trigger dizziness or seisures. We can accommodate them by using the prefers-reduced-motion media feature.

To understand how this demo works, you'll need to disable or reduce animations for your operating system.

  • Windows: Settings →Ease of Access →Display → Show animations in Windows.
  • macOS:System Preferences →Accessibility →Display →Reduce motion
  • iOS:Settings →General →Accessibility →Reduce Motion
  • Ubuntu: Using Gnome Tweak Tool go to General and enable Animations

If you've indicated that you prefer reduced motion in your operating system, the animation below cycles through the color wheel. Otherwise, you'll see a skew, scale, and rotate transformation.
