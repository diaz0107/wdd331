Understanding the :root pseudo-class

The :root pseudo class affects the root element of a document. For HTML documents, that's the html element. For SVG documents, it's the svg element.

In this demo, root.css is linked from both the HTML document and the SVG document included in the iframe. As a result, the root element for both documents have a 10 pixel, dotted magenta border. However, when an SVG document is embedded within an HTML document, it is no longer a root element.
