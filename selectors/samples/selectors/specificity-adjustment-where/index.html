:where(), the Specificity-Adjustment Selector

CSS Master

In this demo, the selector used to style the navigation links above ([id=demo] nav a) has a lower specificity than the selector used to match all links ([id=demo] a:not( :hover )). Here's an example of a jump link that matches the latter selector.

However, adding :where() to our selector changes things: [id=demo] a:where( :not( :hover ) ). Its specificity score remains the same, but :where() makes it so that the declarations attached to the less-specific selector get applied.
