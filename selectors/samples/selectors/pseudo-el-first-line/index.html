This text is enclosed within <q> tags. Firefox includes the opening punctuation in the first line's styling. Chromium-based browsers, on the other hand, do not.

Adding a <br> or <hr> tag
to a line break forces a first line. Styles are limited to characters that precede the forced line break or horizontal rule.

Generated content can become a first line. In this example, we've added content using ::before, but styled it using display: block to force a first line. Unfortunately, this does not yet work in Firefox.
