Understanding the background-blend-mode Property

img { background-blend-mode: normal }

The background-blend-mode property combines background colors and images. In this case, main has two, layered background images formed by linear gradients.

Even though we've only indicated one value for background-blend-mode here. It's possible to set a different value for each layer of a background (e.g.: background-blend-mode: multiply, difference).
