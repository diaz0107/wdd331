Using Multiple Mask Images

This demo only works when the browser can create an origin. You may need to run a local server for this demo.

The mask-image property supports multiple mask images, much like its background-image counterpart. This demo uses square.svg and circle.svg.

You can shape how multiple mask images combine to reveal portions of a layer using the mask-composite property.
