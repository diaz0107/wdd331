Using the clip-path Property: ellipse() function

The clip-path property accepts a basic shape or the URL of an SVG clip path.There are five basic shape functions defined by the CSS Shapes specification:

  • inset() (used to create rectangles)
  • circle()
  • ellipse()
  • polygon()
  • path() (which must be an SVG path data string)
