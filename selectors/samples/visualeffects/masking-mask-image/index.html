Creating a Mask with mask-image

This demo only works when the browser can create an origin. You may need to run a local server for this demo.

Masking is likely familiar to you if you’ve ever worked with graphics editors such as Photoshop, Sketch, or Glimpse. It’s an effect created when the painted areas of one layer reveal portions of another layer.

mask-image behaves much like background-image, and also supports multiple image masks.
