Changing an Element’s Position With translate()

transform: translate( 0, 0 )

Adjust the slider to translate the object below. Translations are relative to the element's starting position. The translate() function accepts up to two arguments. The first argument is the X offset value; the second is the Y offset. If there isn't a second argument, its value is 0. Using translate( 200px ) or translate( 200px, 0 ) is equivalent to translateX( 200px ) and translate( 0, 200px ) is the equivalent of translateY( 200px ).
