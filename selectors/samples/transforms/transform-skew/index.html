Skewing Elements With skew()

transform: skew( 0, 0 )

The skew() function accepts up to two arguments: an x-axis skew amount and a y-axis skew amount. If you pass a single argument, it's the same as using skewX() — only points on the x-axis get skewed. Adjust the slider to skew the object below.
