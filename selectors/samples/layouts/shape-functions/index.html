Using Shape Functions

Shapes are a type of CSS value, like length, angle, or color units. Although used here with the shape-outside property, shape values also apply to the clip-path and offset-path properties.

This is an example of using the circle() shape function with the shape-outside property. We also need to float the shape to the start or end of a line in order for the text to wrap around the shape.
This is an example of using the ellipse() shape function with the shape-outside property. In this example, we are floating the element to the end of the line, so the text wraps around the starting inline area of the shape.
Using the inset() shape function creates a rectangular shape. Here too, the shape is floated to the start of the inline direction. In this instance, round 1rem gives the rectangle rounded corners.

We can also use the polygon() shape to create multi-sided shapes such as triangles and hexagons. A polygon can be any shape, as long as that shape has flat sides.

Of course, if the text is not long enough, it may not wrap around all sides of the shape.
