Using Images as Shapes: Gradients With Alpha Transparency

Beautiful Surf

CSS Gradients are also images. That means we can use them with shape-outside.

In this demo, the background image of main::before is a linear gradient, at a -45 degree angle. The image for shape-outside also uses a linear gradient that's positioned at the same angle.

When an image has partial transparency — an alpha channel value between 0 and 1 — content flows around the fully transparent areas of the image. Partially transparent areas of the image are treated as opaque areas.

However, it's possible to change the point at which the content flows by using the shape-image-threshold property.

Also note here that text-align: justify can make text wrap more tightly around a shape. However, it can also stretch letter and word spacing within lines of text. That may not look so great.

The value of shape-image works independently of the value of the background-image gradient. In some cases, however, you may make a design choice ot keep them in sync. This is a perfect case for using custom properties.
