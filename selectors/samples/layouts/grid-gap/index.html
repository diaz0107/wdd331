Adding Gutters With gap

Grid items do not have space between them by default. Instead we need to add gaps or gutters using the column-gap and row-gap properties. We can also use the gap shorthand property.

For compatibility with older browsers, you can still use the grid-column-gap and grid-row-gap properties, or the grid-gap shorthand.

Related specification: CSS Grid Layout Module.
