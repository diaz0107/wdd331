auto-fit Versus auto-fill

The first grid in this example uses grid-template-columns: repeat(auto-fill, minmax(10rem, 1fr)) while the second uses grid-template-columns: repeat(auto-fit, minmax(10rem, 1fr)). Use Firefox' Developer Tools in order to view this demo. Its grid inspection tools are currently the state of the art.

  • auto-fit fits as many grid items as it can within a track line, and collapses empty tracks.
  • auto-fill fits as many grid items as it can within a track line, but does not collapse empty tracks.

Related specification: CSS Grid Layout Module Level 1.
